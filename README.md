# Aggie Fundraisers

Aggie Fundraisers is a centralized place for UC Davis student organizations to post fundraising events so students can quickly discover what’s fundraising events nearby, save on purchases through fundraiser deals, and support campus groups.

Tech Stack: Next.js (frontend), Node.js + Express (backend), and MongoDB Atlas (database) Note: UI is still being updated

**Live site:** https://aggie-fundraisers.vercel.app/
